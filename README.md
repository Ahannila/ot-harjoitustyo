# Ohjelmistotekniikka, harjoitustyö

## Projekti


## Asennus

1. Install dependencies by:

```bash
poetry install
```

2. Build the necessary files:

```bash
poetry run invoke build
```

3. Start the program:

```bash
poetry run invoke start
```
